# Results
 * Analysis of the Competitive Pokemon meta
 * Trends between most used Pokemon
 * The most commonly used type is Grass type being used 102.84 percent of the time. The reason why the usage is over 100 is because you can have more than ine Grass Pokemon on your team at a time.
 * The most used generation is generation 5 or Pokemon Black and White released in 2011
